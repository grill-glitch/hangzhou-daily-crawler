## 技术细节

### 为何放弃 EPUB

- JavaScript 禁用 → MathJax 失效
- imgmath 方案需完整 LaTeX 环境，维护成本高
- 结论：专注 HTML+PDF

### 当前配置

- `src/conf.py`: 仅 `sphinx.ext.mathjax`
- PDF: XeLaTeX 构建，224 页
- HTML: MathJax 渲染公式
